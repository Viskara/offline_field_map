GPS & Location Permissions

Why does the browser keep asking?
Browsers cannot grant permanent geolocation — it's a browser security boundary, not something this app controls. There is no SSL certificate workaround for a local HTML file.

To minimise prompts:
1. Install as a PWA (use the Install App button in the toolbar) — installed PWAs remember permissions per-app.
2. On Android: open in Chrome → tap ⋮ → Add to Home Screen.
3. On desktop: open in Chrome/Edge → address bar install icon → Install.
4. After installing, tap the GPS button once and choose Allow while using app — the PWA shell remembers it.
